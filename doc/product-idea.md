# Идея продукта: ИИ-ассистент для консультаций клиентов

## Описание

Разработка ИИ-ассистента в виде телеграм-бота для проведения первичных консультаций клиентов по услугам компании.

## Основная задача

Бот должен выполнять следующие функции:
- Отвечать на вопросы клиентов о компании и её услугах
- Уточнять потребности и проблемы клиентов
- Предлагать подходящие услуги компании для решения выявленных проблем

## Технические особенности

### Платформа
- **Telegram Bot** — основная платформа взаимодействия с клиентами

### Подход к работе с данными
- Информация о компании и услугах подготавливается вручную
- Данные добавляются в системный промпт LLM
- Бот использует LLM для генерации ответов на основе предоставленной информации

## Целевая аудитория

Потенциальные клиенты компании, которые:
- Ищут информацию об услугах
- Хотят получить первичную консультацию
- Нуждаются в помощи с определением подходящего решения

## Ожидаемые результаты

- Автоматизация первичной консультации клиентов
- Снижение нагрузки на менеджеров по продажам
- Улучшение клиентского опыта за счет быстрых ответов 24/7
- Квалификация лидов до передачи менеджерам
